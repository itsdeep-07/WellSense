# 🧠 WellSense — Student Mental Health Prediction

> A machine learning project for predicting depression risk in university students.  
> University ML Assessment Project

---

## 📌 Overview

**WellSense** is a machine learning classification system that predicts whether a student is at risk of depression based on demographic and academic features. It trains and compares 7 classical ML models and deploys the best performer as an interactive Streamlit web application.

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/WellSense.git
cd WellSense
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Download dataset
- Go to: https://www.kaggle.com/datasets/shariful07/student-mental-health
- Download and place CSV in `data/Student Mental health.csv`

### 4. Run the Jupyter Notebook
```bash
jupyter notebook notebooks/WellSense_ML_Pipeline.ipynb
```
Run all cells — this trains models and saves artifacts to `models/` and charts to `assets/`.

### 5. Launch Streamlit App
```bash
streamlit run app.py
```

## 📊 Pipeline Phases

| Phase | Description |
|-------|-------------|
| 1 — EDA | Load data, explore distributions, correlations, demographics |
| 2 — Preprocessing | Label encoding, StandardScaler, SMOTE, PCA |
| 3 — Model Building | Train 7 models, collect predictions |
| 4 — Evaluation | Accuracy, Precision, Recall, F1, ROC-AUC, Confusion Matrix |
| 5 — Optimisation | GridSearchCV, K-Fold Cross Validation |
| 6 — Save | Export best model + artifacts with Joblib |
